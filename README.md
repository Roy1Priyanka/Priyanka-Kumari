# Coding Test – Tandemloop

This repository contains my solutions for the Tandemloop First Screening Test.  
All four programs are written in Java and placed in separate files as instructed.

## Files
- **Problem1.java** – Basic calculator class (add, subtract, multiply, divide)
- **Problem2.java** – Prints the first N odd numbers
- **Problem3.java** – Generates an odd-number series using the “nearest odd” rule
- **Problem4.java** – Counts how many numbers in a list are divisible by numbers from 1 to 9

## How to Run

### Compile
javac Problem1.java
javac Problem2.java
javac Problem3.java
javac Problem4.java

## Run
java Problem1
java Problem2
java Problem3
java Problem4

## Tech Used
- Java (OpenJDK 25)
